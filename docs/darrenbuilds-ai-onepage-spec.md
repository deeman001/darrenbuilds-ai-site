# DarrenBuilds.ai … One Page Founder Site Spec (for Claude Code)

## Goal
Design and implement a single page founder site for **darrenbuilds.ai** that feels **minimal**, **confident**, and **modern** … with quiet authority and subtle forward momentum.

Audience … founders, operators, dev teams, partners, potential acquirers, future hires.

Tone … understated, grounded, slightly energized … never hypey.

---

## Core Principles
• Whitespace first … let the page breathe  
• Typography over decoration  
• High contrast … black, white, neutrals  
• Motion is subtle … never distracting  
• The portrait does the heavy lifting … the site supports it  

---

## Page Structure

### 1) Hero Section (Key Moment)

**Layout** … two columns desktop … stacked mobile.

Left column … copy:

H1 … **Darren Builds**  

Subhead …  
**I build businesses and software. From brick and mortar to AI powered systems.**

Primary CTA … **See what I’m building**  
(Scroll to Projects section)

Secondary link … Instagram  
https://instagram.com/darren_builds

Right column … portrait:

Use the close up black and white portrait … hand on chin … slight smirk.

Requirements:
• Eyes near top third  
• No border  
• No heavy shadow  
• No flashy effects  
• Matte look  

Optional subtle motion … gentle scale on scroll (3 to 5 percent max).

---

### 2) The Builder Arc

Headline … **The Builder Arc**

Use this exact copy:

I started coding at 11.  
Computer science was always home.  

Then I built Finley’s Barbershop … an idea that became 18 years of building and refining a multi location brand.  

After leading its growth and transition, I returned to software … fully focused, at a time when AI is redefining what’s possible.  

Same builder. Different tools.  

**This might be the most exciting time to build.**

Formatting:
• Preserve line breaks  
• Add extra whitespace before the final bold line  
• Let that last sentence breathe  

---

### 3) What I’m Building Now

Headline … **What I’m Building Now**

Render as clean rows or simple cards:

PaletteCheck … AI powered brand compliance.  
PolicyPhile … AI policy Q&A for teams.  
Social API … API driven distribution infrastructure.  
Experiments … Always something in progress.

Optional status badge … Building … Beta … Live

Keep it minimal.

---

### 4) How I Build

Headline … **How I Build**

• Systems over features  
• Distribution matters  
• Delete more than you add  
• Real world experience informs better software  

No icons needed … clean bullet style.

---

### 5) Footer

Email … hello@darrenbuilds.ai  
Links … X … Instagram … LinkedIn  

Optional small copyright line.

---

## Visual Direction

Background … white or near white  
Primary text … near black  
Secondary text … charcoal  
Borders … subtle light gray  
Optional accent … muted steel blue  

Typography:
• Large refined headline  
• Clean sans serif body  
• Generous vertical spacing  
• Max width 1040 to 1200px  

---

## Accessibility

• Semantic headings  
• Alt text for portrait … “Black and white portrait of Darren”  
• WCAG AA contrast  
• Respect prefers reduced motion  

---

## Performance

• Optimize hero image  
• Use responsive image sizes  
• Use webp or avif  
• Keep it fast  

---

## Tech Preference

Option A … Next.js deployed on Vercel or Cloudflare Pages  
Option B … Plain HTML and CSS static page  

Single route … /  
Anchor links only.

---

## Deliverable

One polished, responsive, minimal single page site  
Using the close up portrait in the hero  
With calm confidence and subtle forward momentum.
